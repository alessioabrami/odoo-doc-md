# Passare da Community a Enterprise¶

In base all’installazione attuale, esistono vari modi per aggiornare la versione community. In ogni caso, ecco le linee guida di base:

  * Esegui il back-up del database community

  * Arresta il server

  * Installa il modulo erb_enterprise

  * Riavvia il server

  * Inserisci il codice dell’abbonamento Odoo Enterprise.




## Su Linux, utilizzando un programma di installazione¶

  * Esegui il back-up del database community

  * Interrompi il servizio Odoo
        
        $ sudo service odoo stop
        

  * Installa il .deb enterprise (dovrebbe essere installato sul pacchetto community)
        
        $ sudo dpkg -i <path_to_enterprise_deb>
        

  * Aggiorna il database al pacchetto Enterprise utilizzando
        
        $ python3 /usr/bin/odoo-bin -d <database_name> -i web_enterprise --stop-after-init
        

  * Dovresti essere in grado di collegarti all’istanza Odoo Enterprise utilizzando il solito metodo di identificazione. In seguito, puoi collegare il database all’abbonamento Odoo Enterprise inserendo il codice ricevuto via e-mail nel modulo




## Su Linux, utilizzando il codice sorgente¶

Ci sono molti modi per lanciare il server quando utilizzi codici sorgente e probabilmente avrai il tuo preferito. Potrebbe essere necessario adattare le sezioni al flusso di lavoro comune.

  * Arresta il server

  * Esegui il back-up del database community

  * Aggiorna il parametro `--addons-path` del comando di lancio (see [Installare dalla sorgente](source.html))

  * Installa il modulo web_enterprise utilizzando
        
        $ -d <database_name> -i web_enterprise --stop-after-init
        

Potrebbe volerci del tempo, in base alla dimensione del database.

  * Riavvia il server con il percorso dei componenti aggiuntivi aggiornato del punto 3. Dovresti essere in grado di collegare la tua istanza. In seguito, puoi collegare il database all’abbonamento Odoo Enterprise inserendo il codice ricevuto via e-mail nel modulo




## Su Windows¶

  * Esegui il back-up del database community

  * Disinstalla Odoo Community (utilizzando Disinstalla eseguibile nella cartella di installazione) - PostgreSQL non verrà disinstallato

  * Lancia il programma di installazione di Odoo Enterprise e segui i vari step. Quando scegli il percorso di installazione, puoi configurare la cartella dell’installazione Community (la cartella contiene ancora l’installazione di PostgreSQL). Togli la spunta dalla casella `Avvia Odoo` al termine dell’installazione

  * Aggiorna il database Odoo tramite una finestra di comando, utilizzando il seguente (dal percorso di installazione di Odoo, nella sottocartella del server)
        
        $ ..\python\python.exe odoo-bin -d <database_name> -i web_enterprise --stop-after-init
        

  * Non è necessario avviare manualmente il server, il servizio è in esecuzione. Dovresti essere in grado di collegarti all’istanza Odoo Enterprise utilizzando il solito metodo di identificazione. In seguito, puoi collegare il database all’abbonamento Odoo Enterprise inserendo il codice ricevuto via e-mail nel modulo
