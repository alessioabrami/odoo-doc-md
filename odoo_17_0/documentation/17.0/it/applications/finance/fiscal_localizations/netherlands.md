# Olanda¶

## XAF Export¶

With the Dutch accounting localization installed, you will be able to export all your accounting entries in XAF format. For this, you have to go in Accounting ‣ Reporting ‣ General Ledger, you define the entries you want to export using the filters (period, journals, …) and then you click on the button **EXPORT (XAF)**.

## Dutch Accounting Reports¶

If you install the Dutch accounting localization, you will have access to some reports that are specific to the Netherlands such as :

  * Conto economico

  * Tax Report (Aangifte omzetbelasting)

  * Intrastat Report (ICP)
